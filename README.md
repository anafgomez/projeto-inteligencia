# Inteligenc.ia
Inteligenc.ia é uma plataforma desenvolvida para ajudar nos estudos utilizando inteligência artificial. A aplicação faz integração com a API da OpenAI para gerar sugestões, responder perguntas e oferecer auxílio inteligente durante o processo de aprendizado.

# Funcionalidades
Assistente de Estudos: Utilize inteligência artificial para gerar respostas, resumos e sugestões relacionadas ao conteúdo estudado.
Interface Amigável: Design focado na usabilidade, com fundo preto para uma experiência visual confortável e detalhes em vermelho para destacar interações importantes.
Histórico de Sugestões: Armazene as interações anteriores para referência futura.
Tecnologias Utilizadas
React: Para a construção da interface do usuário.
TypeScript: Para tipagem estática e maior segurança no desenvolvimento.
OpenAI API: Para fornecer respostas inteligentes e interações automáticas com o usuário.

# Como Rodar o Projeto
- Clone o repositório:
git clone https://github.com/seu-usuario/inteligencia.git
- Instale as dependências:
npm install
- Inicie o servidor local:
npm start

# Integração com a API da OpenAI
Este projeto utiliza a API da OpenAI para fornecer funcionalidades inteligentes. No entanto, a integração com a API pode não funcionar corretamente se os créditos de uso da API estiverem esgotados. Isso pode resultar em erros ao tentar gerar respostas ou sugestões.

Se você encontrar problemas relacionados à API (como o erro 429 Too Many Requests), é provável que os limites de uso tenham sido atingidos. Para verificar isso:

Acesse o painel de controle da OpenAI: OpenAI Dashboard
Verifique o uso e, se necessário, adicione créditos para continuar utilizando a API.

Aviso
Uso da API: Tenha em mente que a OpenAI possui uma cota limitada de uso para a API. Se você estiver utilizando o projeto sem ter créditos ativos, algumas funcionalidades poderão não responder corretamente.
Limitações: Algumas funcionalidades podem estar limitadas devido à política de uso de API da OpenAI. Por favor, confira as permissões e limites de seu plano de API antes de utilizar em produção.
